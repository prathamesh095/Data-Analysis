
# Getting Speaker Insights Using TED Dataset

This project involves analyzing TED Talk data to extract valuable insights about speakers and their presentations. By exploring various attributes such as speaker profiles, talk durations, themes, and audience engagement, we aim to uncover patterns and trends that contribute to the success of TED Talks. The analysis leverages data visualization and machine learning techniques to identify key factors that influence the popularity and reach of talks, providing a deeper understanding of what makes a TED Talk impactful.


## Questions Covered in This Project

1. **Finding the Most Popular TED Talks**  
   - Identifying TED Talks with the highest view counts to determine which talks have resonated most with audiences.

2. **Finding the Most Popular TED Talks Speaker (in terms of number of talks)**  
   - Analyzing which speakers have delivered the most TED Talks and exploring the reasons behind their frequent appearances.

3. **Month-wise Analysis of TED Talk Frequency**  
   - Investigating how the frequency of TED Talks varies across different months of the year to identify any seasonal patterns.

4. **Year-wise Analysis of TED Talk Frequency**  
   - Examining the trends in TED Talks over the years to understand how the platform has evolved in terms of content delivery.

5. **Finding TED Talks of Your Favorite Author**  
   - Searching for TED Talks by a specific speaker to cater to users interested in talks by their preferred authors.

6. **Finding TED Talks with the Best View to Like Ratio**  
   - Determining which TED Talks have the best engagement rates, based on the ratio of views to likes.

7. **Finding TED Talks Based on Tags (like climate)**  
   - Filtering TED Talks based on specific tags or themes, such as climate, to explore talks centered around particular topics.

8. **Finding the Most Popular TED Talks Speaker (in terms of number of views)**  
   - Identifying speakers whose talks have garnered the highest total views, highlighting the most impactful communicators on the TED platform.

## Setup Instructions

1. **Clone the Repository**
First, clone this repository to your local machine using the following command:

```bash
git clone https://github.com/prathamesh095/Data-Analysis/tree/main/Speaker_Insights_Using_TED
cd Speaker_Insights_Using_TED

```

2. **Create a Virtual Environment**
It's recommended to create a virtual environment to manage your project dependencies:

```bash
python -m venv venv
source venv/bin/activate  # On Windows, use `venv\Scripts\activate`

```

3. **Install Dependencies**
Ensure that you have Python 3.x installed on your system. Then, install the required Python libraries using `pip`:

```bash
pip install pandas numpy matplotlib seaborn jupyter
```

The required dependencies are:
- **pandas**: For data manipulation and analysis.
- **numpy**: For numerical operations.
- **matplotlib**: For data visualization.
- **seaborn**: For statistical data visualization.
- **Jupyter Notebook** (optional): For interactive data exploration and analysis.

You can install all these dependencies in one go using the command provided above.

4. **Download the Dataset **
Make sure to download the TED Talks dataset and place it in the appropriate directory within the project folder. You can download the dataset from this [![GitHub](https://img.shields.io/badge/GitHub-181717?style=for-the-badge&logo=github&logoColor=white)](https://github.com/prathamesh095/Data-Analysis/blob/main/Speaker_Insights_Using_TED/ted_data.csv)

5. **Run the Project**
Execute the main script to start the analysis:

```bash
python main.py
```


**Ensure that all dependencies are installed using the requirements.txt file provided. If you want to add additional libraries, simply update the requirements.txt file and re-run the installation command.**
